Врублевский Виталий

Я занимаюсь разработкой ПО, администрированием серверов выполнял такие задачи как создание REST API, оркестрация, виртуализация. Для этого я решал следующие задачи:

  • Разработка backend на python + django(DRF) - Часто приходилось расширять штатное поведение системы, переопределять пользовательские модели и валидаторы, добавлять собственные решения для вопросов информационной безопасности, работал с очередями и их брокерами ;)
  • Использовать Bootstrap 4 для верстки, дизайна страниц и VUE для взаимодействия с API и пользовательскими действиями, работал с npm, yarn. Также мне очень нравятся статические генераторы сайтов такие как Next.js, Jekyll ;)
  • Использовал Docker для контейнеризации и создания микро сервисной архитектуры, строил отказоустойчивые docker-swarm кластеры
  • Занимался логированием приложений и всех сервисов внутри кластера для этого я использовал logspout + ELK stack
  • Использовал Ansible для описания, настройки и создания серверов, писал также тесты для playbook (Molecule + docker)
  • Использовал Elasticsearch для полнотекстового поиска, масштабировал и делал ее отказоустойчивой ;)

Мои ближайшие цели: Изучить golang, javascript более углубленно, не зацикливаться на python, что бы смотреть на одни и теже вещи под разным углом. Хотел бы освоить систему мониторинга Prometheus и оркестрации Kubernetes на уровне использования в "prod". Получить более глубокие знания в области сетей. Развивать устную Английскую речь ;)
